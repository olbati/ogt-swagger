### Tech
This example uses a number of open source projects to work properly:

* [spring]
* [hibernat validator]
* [mongodb]
* [swagger2]

### Requirements

- Install a mongodb database.
- A smtp server.

### Run

```sh
$ mvn clean install
$ mvn spring-boot:run
```
